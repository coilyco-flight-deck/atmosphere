# joke-apps

**What** - a deliberately small, silly app. The Flushes slot (a bathroom check-in app that is real and works).

**Why it fits the Atmosphere** - it is the purest demonstration of the thesis. Someone defined a `flushes` lexicon and shipped without asking anyone. A joke app is a permissionless-app proof in one afternoon.

**Why it might be for Kai** - a joke app is the cheapest possible end-to-end run through the whole stack: define a lexicon, write records, build a tiny App View, publish. It is a learning vehicle disguised as a bit.

**Open questions**

- What the bit is. A coily-themed check-in, an Eco-server status toy, a galaxy-gen daily roll.
- Best used as the warm-up that teaches the lexicon-to-App-View loop before a serious build.
