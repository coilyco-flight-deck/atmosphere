# categories

The full AT Protocol implementation-category space, mapped so nothing gets forgotten. Each category is a slot. Some slots already have a dedicated idea file, some have a fresh seed file, a few are deliberately parked.

Source of the category list: `reference/atproto-ecosystem.md`.

- **Microblogging** - Kai already builds on Bluesky directly. No idea file needed, it is the substrate.
- **Clients** - alternative front doors for Bluesky. Seed: [clients](clients.md).
- **Long-form / publishing** - covered by [whitewind-blog](whitewind-blog.md).
- **Link aggregation / forums** - covered by [front-page](front-page.md).
- **Photo** - parked. Kai is not interested as a consumer surface. The only live angle is the Kapwing integration, covered by [kapwing-atproto-media](kapwing-atproto-media.md).
- **Video / streaming** - parked, same reasoning as photo. See [kapwing-atproto-media](kapwing-atproto-media.md).
- **Reviews / tracking** - Letterboxd-for-X shape. Seed: [reviews-tracking](reviews-tracking.md).
- **Events** - RSVPs, calendars. Seed: [events](events.md).
- **Dev tools** - Tangled and the developer-workflow surface. Seed: [dev-tools](dev-tools.md).
- **Chat / messaging** - covered by [chat](chat.md) and [atproto-messaging](atproto-messaging.md).
- **Art / creative** - draw-in-browser, custom emoji. Seed: [art-creative](art-creative.md).
- **Profile / link-in-bio** - Linktree-shaped. Seed: [profile-link-in-bio](profile-link-in-bio.md).
- **File storage** - covered by [atfile-storage](atfile-storage.md).
- **Moderation / labelers** - composable moderation. Seed: [moderation-labelers](moderation-labelers.md).
- **Infrastructure / plumbing** - covered by [jetstream-parser](jetstream-parser.md), [constellation-backlinks](constellation-backlinks.md), [infra-primitives-overlap](infra-primitives-overlap.md).
- **Joke apps** - the Flushes slot. Seed: [joke-apps](joke-apps.md).

Strategic lens over all of it: [atproto-x-my-streams](atproto-x-my-streams.md). The first real app is [topic-feeds](topic-feeds.md).
