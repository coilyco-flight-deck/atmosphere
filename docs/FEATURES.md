# Features

Inventory of what ships today.

## Ideas

`ideas/` - one markdown file per topic-feed candidate or atproto experiment. Currently seeded with eight ideas. No structure beyond "one file per idea."

## Packages

Empty. Reserved for shared atproto plumbing: lexicon definitions, identity and auth, PDS client, Jetstream firehose consumer.

## Apps

Empty. Reserved for actual Atmosphere apps. As an app proves itself and stops sharing churn with the rest, it graduates out into its own repo.
