# constellation-backlinks

**What** - a global backlink index, Constellation-shaped. Answers "who replied to, quoted, or liked this record across the whole network." Kai flagged it as useful and worth coming back to.

**Why it fits the Atmosphere** - the firehose carries every record on the network. An index that inverts it (record -> everything pointing at it) is a backlink graph. Constellation and microcosm already do this.

**Open questions**

- Run or query an existing index vs build one. Building one means ingesting the whole network, which is heavy.
- What Kai would do with backlinks. This connects to [wiki-walker](wiki-walker.md) - backlinks are edges, and a backlink index over Kai's own records is a knowledge-graph feed.

Status: think about it later. Useful, not urgent.
