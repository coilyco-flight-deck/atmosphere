# events

**What** - an events surface, Smoke Signal shape. Events, RSVPs, calendars as atproto records.

**Why it fits the Atmosphere** - an event is a record, an RSVP is a record pointing at it. A custom lexicon plus an App View that renders calendars.

**Why it might be for Kai** - the Sirens Eco community runs on Discord. An atproto events surface could give that community portable, non-Discord-locked event records. Also ties to Eco cycle-prep announcements.
